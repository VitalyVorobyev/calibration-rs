# Task Reports

Each completed backlog task must add one concise report file in this directory.

Naming convention:

- `YYYY-MM-DD-<task-id>-<slug>.md`

Example:

- `2026-03-07-M1-T01-scheimpflug-module-shape.md`

Required sections:

1. Scope
2. Files changed
3. Validation run
4. Follow-ups / risks
